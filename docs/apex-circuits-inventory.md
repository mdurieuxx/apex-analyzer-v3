## Notes

- **Port WS = configPort + 3** (règle Apex Timing universelle)
- **3 hôtes possibles** : `www.apex-timing.com` · `live.apex-timing.com` · `live-data.apex-timing.com`
- **Port 0** = à découvrir par scan (SEED_CIRCUITS_LIVE, port non confirmé)
- **Port -1** = scan effectué, port non trouvé (circuit peut-être inactif)
- **✓** = port WS joignable lors du dernier test
- **✗** = port WS injoignable ou non trouvé
- **0** = jamais testé
